# PODNAME: Authen::Krb5::Keytab
# ABSTRACT: A Kerberos 5 keytab

=method C<add_entry(entry)>

Adds I<entry> to the keytab.

=method C<remove_entry(entry)>

Removes I<entry> from the keytab.

=method C<get_name()>

Returns the name of the keytab.

=method C<get_entry(principal[, kvno, enctype])>

Returns an L<Authen::Krb5::KeytabEntry> object representing an entry in the keytab matching C<principal> and optionally C<kvno> and C<enctype>.

=method C<start_seq_get()>

Returns a cursor that can be passed to L</next_entry> to read in turn every key in the keytab.

=method C<next_entry(cursor)>

Returns the next entry in the keytab as an L<Authen::Krb5::KeytabEntry> object.

=method C<end_seq_get(cursor)>

Perform cleanup operations after L</next_entry> and invalidates C<cursor>.
