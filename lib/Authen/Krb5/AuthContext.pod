# PODNAME: Authen::Krb5::AuthContext
# ABSTRACT: Kerberos 5 auth_context object.

=method C<new()>

Allocates memory for a new Authen::Krb5::AuthContext object and returns it.

=method C<setaddrs(localaddr,remoteaddr)>

Sets the local and remote addresses for the C<AuthContext> object.
C<localaddr> and C<remoteaddr> are L<Authen::Krb5::Address> objects, usually of type C<ADDRTYPE_INET>.

=method C<getaddrs()>

Returns a list containing the local and the remote address of the C<AuthContext> object.

=method C<setrcache(rc)>

Sets the replay cache for auth_context.
C<rc> is a L<Authen::Krb5::Rcache> object generated by L<Authen::Krb5/get_server_rcache>.

=method C<getkey()>

Retrieves the session key as an L<Authen::Krb5::KeyBlock> object.
