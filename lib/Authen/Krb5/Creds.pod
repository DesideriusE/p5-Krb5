# PODNAME: Authen::Krb5::Creds
# ABSTRACT: Object representing a credential.

=method C<starttime()>

Returns the C<starttime> time property of the credential.

=method C<authtime()>

Returns the C<authtime> time property of the credential.

=method C<endtime()>

Returns the C<endtime> time property of the credential.

=method C<renew_till()>

Returns the C<renew_till> time property of the credential.

=method C<server()>

Returns the name of the service principal the credential is for.

=method C<client()>

Returns the client principal name (will usually be identical for all credentials in a credential cache).

=method C<ticket()>

Returns the L<Authen::Krb5::Ticket> for this credential.

=method C<keyblock()>

Returns the C<keyblock> of the credential.
